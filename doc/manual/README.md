# Local Photo Contest 利用マニュアル

Local Photo Contest システムの利用マニュアル集です。

---

## マニュアル一覧

### 1. [参加者向けマニュアル](./participant_guide.md)

フォトコンテストに参加する方向けのガイドです。

**対象**: 一般ユーザー（コンテスト参加者）

**内容**:
- アカウント登録・ログイン
- コンテストの探し方
- 写真の応募方法
- 応募作品の管理
- 投票の仕方
- コンテンツ審査について

---

### 2. [主催者向けマニュアル](./organizer_guide.md)

フォトコンテストを主催・運営する方向けのガイドです。

**対象**: 主催者（オーガナイザー）

**内容**:
- コンテストの作成・管理
- 応募作品の管理
- コンテンツモデレーション
- 撮影エリア・スポットの設定
- 審査と結果発表

---

### 3. [審査員向けマニュアル](./judge_guide.md)

コンテストの審査員として評価を行う方向けのガイドです。

**対象**: 審査員（主催者から招待された方）

**内容**:
- 招待の受諾方法
- 審査員ダッシュボードの使い方
- 作品の評価方法
- 評価基準とスコア入力
- フィードバックコメントの書き方
- 評価の編集

---

### 4. [システム管理者向けマニュアル](./admin_guide.md)

システムの管理・運用を行う方向けのガイドです。

**対象**: システム管理者

**内容**:
- ユーザー管理
- カテゴリ管理
- モデレーション設定
- システム監視
- バックアップと復元
- トラブルシューティング

---

## クイックリファレンス

### ユーザーロールと権限

| ロール | 応募 | 投票 | 作品評価 | コンテスト作成 | モデレーション | システム管理 |
|:------:|:----:|:----:|:--------:|:--------------:|:--------------:|:------------:|
| 参加者 | ✅ | ✅ | - | - | - | - |
| 審査員 | ✅ | ✅ | ✅ | - | - | - |
| 主催者 | ✅ | ✅ | - | ✅ | ✅ | - |
| 管理者 | ✅ | ✅ | - | ✅ | ✅ | ✅ |

> 💡 審査員は主催者から招待されることで、特定のコンテストの審査権限を得ます。

### よく使う機能

| やりたいこと | 参照マニュアル |
|-------------|----------------|
| コンテストに応募したい | [参加者向け - 写真を応募する](./participant_guide.md#3-写真を応募する) |
| 応募が非表示になった | [参加者向け - コンテンツ審査について](./participant_guide.md#7-コンテンツ審査について) |
| 審査員の招待を受諾したい | [審査員向け - 招待の受諾](./judge_guide.md#2-招待の受諾) |
| 作品を評価したい | [審査員向け - 作品の評価](./judge_guide.md#4-作品の評価) |
| コンテストを作成したい | [主催者向け - コンテストの作成](./organizer_guide.md#3-コンテストの作成) |
| 審査員を招待したい | [主催者向け - 審査と結果発表](./organizer_guide.md#8-審査と結果発表) |
| 不適切な投稿を管理したい | [主催者向け - コンテンツモデレーション](./organizer_guide.md#6-コンテンツモデレーション) |
| ユーザーに主催者権限を付与したい | [管理者向け - ユーザー管理](./admin_guide.md#2-ユーザー管理) |
| モデレーションが動作しない | [管理者向け - トラブルシューティング](./admin_guide.md#7-トラブルシューティング) |
